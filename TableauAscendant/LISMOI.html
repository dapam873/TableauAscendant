Tableau ascendance

Ce logiciel permet de produire un ou des fichiers PDF de Tableau d'ascendance.

Les Tableaux peuvent être créés en PDF vierge aussi bien que rempli. Les données entrées peuvent être enregistrées.

Instruction d'installation

  1. Si le logiciel a été préalablement installé sur votre ordinateur, vous devez procéder à sa désinstallation selon les procédures habituelles de votre version de Windows.
  2. Le fichier tableauascendant.zip doit être décompressé avant de procéder à l'installation.
  3. Démarrer setup dans le dossier décompressé.

Si vous avez des erreurs d'installation. S'assurez qu'il n'y a pas déjà eu une installation du logiciel dans votre ordinateur, redémarrer l'ordinateur et refaites l'étape 3 et attention le faire à partir du dossier décompresser.
