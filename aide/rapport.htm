Rapport

Si on indique pour le champ SOSA le numéro 0 ou on le laisse vide, le rapport sortira vierge dans le PDF.

Pour sortir les rapports:

⧫ Première étape, choisir le dossier ou sera enregistré les fichiers PDF.

★ Dans le menu en haut de la fenêtre, ⟶ Paramètre ⟶ Dossier PDF et suivre la procédure normale de Windows pour choisir le dossier.

⧫ Deuxième étape, choisir le type de rapport.

★ Une page

• Dans le menu en haut de la fenêtre, ⟶ PDF ⟶ Créer page courante.

∘ Ceci enregistrera, puis affichera le fichier PDF.

∘ Le fichier portera pour nom le numéro de la page.

★ Toutes les pages

• Dans le menu en haut de la fenêtre, ⟶ PDF ⟶ Créer toutes les pages.

∘ Ceci enregistrera, puis affichera le fichier PDF.

∘ Le fichier portera pour nom TableauAscendant.
